# Changelog

All notable changes to Glidex will be documented in this file.

## 0.1.0 - Unreleased

- Add Device Hub and legacy Simulator attachment.
- Add Navigate, Direct Touch, Point, and Edge input modes.
- Support click, long press, drag, swipe, pinch, rotation, and up to five
  Direct Touch contacts.
- Add anchor and active-touch indicators with configurable border visibility.
- Add versioned JSON gesture recording, local storage, file selection, and
  deterministic replay.
- Add diagnostics, input constraints, reconnect controls, and Direct Touch
  keyboard switching.
- Add a small injection and recording replay CLI.
